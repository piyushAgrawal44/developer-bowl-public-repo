## 🧪 Getting Started

To run this project locally, follow these steps:

### 1. Clone the repo

```bash
git clone https://github.com/your-username/developer-bowl.git
cd developer-bowl
```

### 2. Install dependencies

```bash
npm install
# or
yarn install
```

### 3. Run the development server

```bash
npm run dev
# or
yarn dev
```

Visit `http://localhost:3000` in your browser.
